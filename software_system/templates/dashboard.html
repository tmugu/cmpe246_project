Project Description

Security-Focused IoT Monitoring Platform

This project combines an RC522 RFID reader, Raspberry Pi hardware, a Flask backend, and an SQLite database to create a smart access control and attendance platform. Each RFID scan is checked against registered users, turned into an allow or deny decision for the physical system, and stored as a timestamped attendance event. The dashboard lets administrators track activity in one place, manage registered cards, observe access trends, and export records for reporting or submission.
